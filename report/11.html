5. 找出 Oracle 与 MySQL，PostgreSQL 的 SQL 实现语法不同之处，每个数据库各找 5 种差异

Oracle 与 Mysql/PgSql 差异之处
选择工资前三的员工信息
Oracle select * from (
  select * from emp order by sal desc
) where rownum <= 3;
MySql select * from emp order by sal desc limit 3;
PgSql select * from emp order by sal desc limit 3;
选出员工工资中位数
Oracle select median(sal) from emp;
MySql with Numbered as (
  select *, count(*) over () as Cnt,
  ROW_NUMBER() over (order by sal) as RowNum
  FROM emp
) select avg(sal) from Numbered
 where RowNum in ((Cnt+1)/2, (Cnt+2)/2);`
PgSql select percentile_disc(0.5)
within group (order by sal)
from emp;
查看唯一行标识符
Oracle select rowid from emp;
MySql select _rowid from emp;
PgSql select ctid from emp; （ctid 将随着update而改变）
首字母大写，其他字母小写
Oracle select initcap(lower('HELLO')) from dual;
MySql select concat(UPPER(left("HELLO",1)),
    substring(lower('HELLO'), 2, (length('HELLO') - 1)))
from dual;`
PgSql select initcap(lower('HELLO'));
查看当前登录用户
Oracle select user from dual;
MySql select user();
PgSql select user;
